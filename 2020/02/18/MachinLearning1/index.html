머신러닝

###머신러닝 이란?

  • 데이터를 기반으로 학습을 시켜서 예측하게 만드는 기법
  • 인공지능의 한 분야로 컴퓨터가 학습할 수있도록 하는 알고리즘과 기술을 개발하는 분야

  • 통계학,데이터 마이닝, 컴퓨터 과학이 어우러진 분야
    인공지능 -> 머신러닝 -> 딥러닝

  • Data ➡ Model(알고리즘)

    1. _학습 을 통해 기계가 스스로 규칙 을 만들어낸다._

    2. 데이터를 이용하여 특성과 패턴을 학습 하고, 그 결과를 바탕으로 미지의 데이터에 대한 미래결과(값, 분포)를 예측 하는것

머신러닝 종류

  1. 지도학습(Suprevised Learning)
  2. 비지도학습(Unsupervised Learning)
  3. 강화학습(Reinforcement Learning)

  • 지도학습
    데이터에 대한 Label(명시적인 답)이 주어진 상태에서 컴퓨터를 학습시키는 방법으로 분류(Classification)와 회귀(Regression)로 나뉘어진다.

    분류(Classification) :

    1. 미리 정의된 여러 클래스 레이블 중 하나를 예측하는 것
2. 속성 값을 입력, 클래스 값으로 출력하는 모델
3. 이진 분류, 다중 분류 등이 있다.

회귀(Regression) :

1.연속적인 숫자를 예측하는 것.
2. 속성 값을 입력, 연속적인 실수 값을 출력으로 하는 모델
3. 어떤 사람의 교육수준, 나이, 주거지를 바탕으로 연간 소득 예측
4. 예측 값의 미요한 차이가 크게 중요하지 않다.

EX) 스팸 메일 분류, 집 가격 예측
  • 비지도 학습(Unsupervised Learning)
    데이터에 대한 Label(명시적인 답)이 없는 상태에서 컴퓨터를 학습시키는 방법으로, 데이터의 숨겨진 특징, 구조, 패턴을 파악하는데 사용하고 있으며 데이터를 비슷한 특성끼리 묶는 클러스터링(Clustering)과 차원축소(Dimensionality Reduction)등이 있다.
    • EX) 이미지 감색 처리, 소비자 그룹 발견을 통한 마케팅, 손 글씨 숫자 인식
  • 강화 학습(Reinforcement Learning)
    지도학습과 비슷하지만 완전한 답(Label)을 제공하지 않는 특징이 있으며, 기계는 더 많은 보상을 얻을 수있는 방향으로 행동을 학습 하며, 주로 게임이나 로봇을 학습시키는데 많이 사용합니다.

머신러닝(Machine Learning)이 유용한 분야

첫째, 기존 솔루션으로는 많은 수동 조정과 규칙이 필요한 문제

둘째, 전통적인 방식으로는 전혀 해결 방법이 없는 복잡한 문제

셋째, 새로운 데이터에 적응해야하는 유동적인 환경

넷째, 대량의 데이터에서 통찰을 얻어야 하는 문제

머신 러닝 과정

1.문제정의
    - 비지니스 목적 정의
    - 현재 솔루션의 구성 파악
    - 지도 vs 비지도 vs 강화
    - 분류 vs 회귀
2.데이터 수집
    - Filt(CSV, XML, JSON)
    - Database
    - Web Crawler (뉴스, SNS, 블로그)
    - IOT 센서를 통한 수집
    - Survey
    - 케클
    - 공공데이터
    - 크롤링
3. 데이터 전처리
    - 결측치, 이상치
    - 특성공학 (도메인지식 중요)🧀
    - 인코딩(문자 ➡ 숫자)❓
4. 탐색적 데이터 분석(EDA)🧀
    - 특성선택
    - 데이터 전처리의 전략 수립
5. 모델 선택
    - 지도학습
        - 분류 : KNN(k),Decision Tree(max_dep)
        - 하이퍼파라미터 튜닝 📌중요!!📌
6. 학습 : fit
    - train_test_split
7. 평가 : score
    - 분류 : 정확도(Accuracy), 재현율(Recall), 정밀도()
    - 회귀 : 결정계수(R^2)

인코딩 ❓
