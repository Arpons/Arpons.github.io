Node.js의 시작

Node.js에서 간단한 정적 콘텐츠를 보여주는 웹 서버를 실행하는 것에 대해 살펴본다. Node.js 애플리케이션은 대부분 JavaScript 파일로 구성되며, 코드 라이브러리가 들어있는 JavaScript
파일 또는 폴더를 모듈이라고 부른다.

Node.js & npm

Node.js를 설치하게 되면 Node.jsdml 패키지 관리자인 npm 역시 같이 설치 되게 된다. npm은 외부 패키지
관리 역할을 한다. 아래의 표에 자주쓰이는 명령어 들을 조금 나열해 보았다.

명령 설명
npm init node.js 애플리케이션 초기화 및 package.json 파일 생성
npm install node.js 패키지 설치
npm start package.json 파일에 설정되 파일로 node.js 애플리케이션 실행
npm stop 실행 중 인node.js 애플리케이션 종료

예를 들어 npm install <package>를 사용할 경우 –save 또는 –global 명령어를 붙여 사용 할 수 있다.
약어로 -s ,-g로 표기한다.

npm init 명령어를 실행하여 package.Json파일이 생성되면 새로운 패키지을 설치 할 때 마다 이곳에 기록 될 것이다. 이후 npm install 명령어를 실행하여 node_modules폴더를 생성한다.

요청-응답 사이클

  1. 사용자가 웹 브라우저에 URl을 입력하면 ➡ 요청 ➡ Http 요청이 URL에 연결 된 서버에 전달된다.
  2. 서버에서 Http 응답이 텍스트,Json 또는 다른 포맷 형식으로 ➡ 응답 ➡ 사용자의 웹 브라우저에 전달한다.
